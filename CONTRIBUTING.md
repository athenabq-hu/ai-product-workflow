# 贡献指南

欢迎为 AI 多智能体产品开发工作流贡献内容！

## 如何贡献

### 方式一：提交新的 Agent Prompt
- 有更好的 Prompt 模板？提交 PR！
- 请附上 Prompt 适用的场景说明

### 方式二：补充案例
- 有使用这个工作流完成的项目案例？
- 提交 Issue 描述你的项目，我们会加到 CASE_STUDY.md

### 方式三：改进文档
- 发现错误？直接提交 PR 修改
- 文档不够清晰？提出 Issue 描述问题

## 提交规范

### Commit Message 格式
```
[Type] Short description

Types:
- docs: 文档改进
- prompt: Prompt 模板更新
- case: 案例补充
- fix: 错误修复
```

### PR 规范
1. Fork 本仓库
2. 创建新分支（`git checkout -b feature/your-idea`）
3. 提交改动（`git commit -m '[type] your message'`）
4. 推送到你的 Fork（`git push origin feature/your-idea`）
5. 提交 Pull Request
